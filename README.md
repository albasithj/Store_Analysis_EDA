## Store_Analysis_EDA

## Overview
This project performs Exploratory Data Analysis using Python to understand
data patterns, distributions, and relationships.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Work Done
- Data cleaning and preprocessing
- Handling missing values
- Data visualization
- Correlation analysis

## Outcome
Identified key insights, trends, and data quality issues to support
data-driven decision making.
