## Store_Analysis_EDA

## Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on store sales data
to uncover patterns, trends, and insights related to sales performance, customer behavior,
and product categories. The analysis helps in understanding key business drivers and
supports data-driven decision-making.

---

## Project Objectives
- Understand overall sales performance
- Identify high-performing and low-performing products
- Analyze customer purchase behavior
- Detect trends and patterns in sales data
- Support business insights through data visualization

---

## Dataset Information
The dataset contains store-level sales records with features such as:
- Order and transaction details
- Product categories and sub-categories
- Sales and profit values
- Customer and regional information
- Time-based attributes (date, month, year)

Note: The dataset used in this project is for analysis and learning purposes.

---

## Tools & Libraries Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Data Cleaning & Preparation
- Handled missing values
- Removed duplicate records
- Converted data types where required
- Created new features for better analysis
- Filtered and transformed data for visualization

---

## Key Analysis Performed
- Descriptive statistics of sales and profit
- Sales distribution analysis
- Category-wise and sub-category-wise performance
- Customer and region-based sales analysis
- Time-series analysis to identify trends
- Correlation analysis between numerical features

---

## Key Insights & Outcomes
- Identified top-performing product categories based on sales
- Observed seasonal trends impacting sales performance
- Highlighted regions contributing the highest revenue
- Detected products with low profitability
- Provided insights that can support inventory and pricing strategies
